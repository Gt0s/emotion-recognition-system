# Emotion Recognition System

## Description
This project focuses on recognizing human emotions from audio signals using signal processing and machine learning techniques.

## Features
- Audio feature extraction using MFCC  
- Dimensionality reduction using PCA  
- Basic emotion classification approach  

## Technologies Used
- Python  
- MFCC  
- PCA  

## What I Learned
- Signal processing techniques  
- Feature extraction from audio  
- Basic machine learning concepts
